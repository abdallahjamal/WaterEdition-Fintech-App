# 💧 Water Edition - Advanced Fintech App

تطبيق مالي متطور لإدارة حسابات الزبائن والديون بدقة متناهية، مبني بأحدث تقنيات أندرويد.

## ✨ المميزات (Features)
* **إدارة الزبائن:** إضافة وتعديل وحذف بيانات الزبائن.
* **سجل العمليات:** تتبع كامل لعمليات الدفع (Payment) والتعبئة (Fill).
* **الدقة المالية:** استخدام `BigDecimal` لضمان حساب الأرصدة بدون أي أخطاء حسابية.
* **المزامنة الذكية:** نظام تلقائي لإعادة حساب الرصيد (Auto-Sync) يضمن مطابقة السجل للواقع دائماً.
* **تجربة مستخدم متميزة:** واجهات Glassmorphic عصرية مع نظام تشغيل فوري (Instant Splash Screen).

## 🛠 التقنيات المستخدمة (Tech Stack)
* **Language:** Kotlin
* **UI:** Jetpack Compose (Declarative UI)
* **Architecture:** Clean Architecture (Domain, Data, UI Layers)
* **Local Database:** Room Persistence Library
* **Concurrency:** Kotlin Coroutines & Flow
* **Dependency Injection:** Hilt/Dagger
